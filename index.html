Command Line Cheatsheet

BASH

Reference

Basics

Command Description
pwd Prints working directory
ls Lists items within directory
mkdir directoryName Makes a directory called directoryName
cd directoryName Changes location to directoryName
cd ..
cd../..
cd /givenDirectory		 Move to parent directory
Move to grandparent directory
Move to givenDirectory
touch (path) file.type Creates a new file (optional path)
rm -f file.type Deletes file
rm -d directory Deletes directory

Git

Reference

Basics

Command Description
git init Initialize new repository
git add file.type
git add --all		 Add file to staging area
Add all files to staging area
git commit -m "message" Commit changes to the branch with message
git status Shows untracked files and files staged for commit
git log Displays log of all commits and their information

Pushing to Github

Command Description
git branch -M main Renames current branch to main
git remote add origin https://github.com/... Sets origin = remote repository url
git push -u origin main Pushes the branch 'main' to the url 'origin' /
Continue pushing to remote repository afterwards
git remote set-url origin https://github.com/... Resets origin to repository url

Branching

Command Description
git branch Lists all the branches
git branch branch-name Creates branch called branch-name
git checkout branch-name Moves to branch-name branch
git branch -M other-branch Renames current branch to other-branch
git merge branch-name Merges branch-name to the master branch
git checkout master Moves back to master branch
git branch -d branch-name Deletes branch-name branch
git push origin --delete branch-name Deletes branch-name from remote repository

Backtracking

Command Description
git show HEAD Displays log information + committed file changes
git reset HEAD file.type Removes file.type from the staging area
git checkout HEAD file.type Restores file.type to the latest commit
git reset commit-SHA Resets the head to a previous commit with the same SHA
